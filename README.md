# keras-practice

**Goal:** A personal sandbox for mastering Deep Learning using Keras.

## Getting Started
### Prerequisites
* Python 3.8+
* pip or conda

### Installation
```bash
git clone [https://github.com/CTL342/keras-practice.git](https://github.com/CTL342/keras-practice.git)
cd keras-practice
pip install -r requirements.txt
```

## Dataset Citation
This project utilizes the **Cooling Tower Optimization Dataset** for regression and optimization practice.

* **Dataset Name:** Cooling Tower Optimization Dataset
* **Source:** Kaggle
* **Author:** Ziya
* **License:** CC0: Public Domain
* **Description:** Real-time sensor data collected from 2018-2024, tracking environmental factors (outdoor temp, humidity) and system metrics (inlet/outlet temps) to optimize energy efficiency.
